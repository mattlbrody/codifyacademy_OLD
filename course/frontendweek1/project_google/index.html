Step 2: Add CSS to the Top Navigation

Time to add a little style to your navigation! - Lets switch over to main.css we can just begin on line 1.

Call the HTML element Unordered List with the ID of top_nav and give it two CSS properties - float: ; & margin-right: ;. Attach the value of right to the float property and the value of 25px to the margin-right property. We are done with this css selecter, move below the closing ul#top_nav curly bracer.

The next CSS selector we want to call is for every list item inside any unordered list with the ID of top_nav. To your list items we want to set the list-style: ; property to have a value of none, this will remove the little dots that are on list item tags by default.

Next we want to set the display: ; property add a value of inline-block to it, that way your list items will appear in one line. Now we want to change the text-align: ; property value to right. Adjust the font size to 13px and vertical-align to bottom.

Last we want to add a little padding and margin to add a little space between each list item. Set the padding value to 1em for the top, 10px for the right and 0 for the bottom and left. For the margin property we need to add the values of 5px to the top, 5px for the right and 0 for the bottom and left.

Time to mess around with the icons we added. We can style both at once using the .fa class selector. Increase the font-size value to 20px to increase the size of the icons, they are considered font and we can increase the size that easy.

Also add the opactiy: ; property to the .fa selector and give it a value of 0.7 - add the cursor property with a value of defaultand the last property we will adjust will be vertical-align: ;, we need to set the value to top - so the icons will align correctly.

Lets change what happens when a user hovers over these icons, using the pseudo class :hover add the opacity: ; property and set the vaule to 1.0 making it 100% visible as opposed to 70% when the user is not hovering on one of the icons.

A few more to go and your top_nav will look like googles! top navigation The next select we will need is the span element that has the ID of share (your fifth <li> hopefully). Set the cursor: ; property to have a value of default, Give it a property of background-color: ; and a value of #f2f2f2, a nice lightgrey that matches googles.

We need a little padding to increase the size of the background behind the span, call the padding property and set the value to 7px for the top and bottom, and 15px for the left and right. (without it would look something like this Share, not really resembling a button here.

Lets change what happens when a user hovers over this span, using the pseudo class :hover again this time on the span#share selector. We want to change the background-color property to a color of #E1E1E1 - a darker color so a user will know they are hovered over it.

Final selector for the navigation!!! Call the HTML img element selector that has the class of profile_img. First adjust the width: ; & height: ; of the image, add the value of 32px to each property. Add the border-radius property and set it to have a value of 25px - changing your picture to a nice circle. Last we need to adjust the margin for the picture since its off a little bit, give it a value of -5px and it should fall in line.

Step 6: Add CSS to the bottom navigation section

Now that we finished the HTML portion, lets add some properties to some more elements to finish what we started!!! Back to main.css to finish out the project.

First things first, we want to style the div with a ID of bottom_nav attached to it. We want this div to be attached to the bottom permanently, to do this we need to set the position: ; property to absolute, then set the bottom css property to 0 !important.

Now the div will be attached to the bottom of the browser screen! A few more to add to this one - 1st set the width property to 100%. 2nd add a background-color property and set the value to #F2F2F2, and 3rd add the border-top property and set it to equal 1px for the width, solid for the style, and #E4E4E4 for the border color.

Moving on to the next selector - you need to call the unordered list with a ID of left, set the float property equal to left, the margin-bottom property equal to 0 and a padding-left property of 20px.

Next lets style the unordered list with a ID of #right, as you can imagine we want to float this to the right, as before we need to add a margin-bottom property and set he value equal to 0 and last a padding-left: ; property equal to 10px.

Time to style all the list items found in both ul#right and ul#left - for this selector you want to set the display property to inline, add a line-height: ; and set the value to 40px and last remove the list-style but setting this property to none.

Trucking right along, we want to select every a tag inside each List Item found inside both unordered lists of #left, & #right, first add the color property ad set the value to #666. Second reduce the letter-spacing: ; to a value of 0.1. Third adjust the font-size to 13px, and 4th add a margin: ; of 2px for the top, 10px for the left & right, and 0 for the bottom.

We are down to the last two selectors, and the most complicated so get ready...We want to select the unordered list with the ID of right, we want to find all list items inside this, then the anchors inside the li's and finally the :first-child selector added to the anchor tag. To this selector we want to add the padding-right: ; property and set the value to 8px.

For the last selector you will want to call and unordered list with the ID of #left, then every list item inside and then use the :nth-child() pseudo selector on the list item and pass a value of 2 into it, use a comma so you can add a second selector to this call. Just copy the 1st line with out the comma and pass a value of 3 into it.
